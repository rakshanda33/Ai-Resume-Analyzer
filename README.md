# 📄 AI Resume Analyzer

> AI-powered resume analysis built with Python, Streamlit, and Google Gemini.

🔗 **Live Demo:** [your-render-url.onrender.com](https://your-app.onrender.com)

## Features

| Feature | Description |
|---|---|
| 📊 Resume Score | AI scores resume 0–100 with hire verdict |
| ✅ Strengths & Weaknesses | Evidence-based feedback |
| 🎯 ATS Keyword Matcher | Compare resume vs job description |
| ✏️ Bullet Rewriter | Rewrites weak bullets with strong action verbs |
| 📥 Download Report | Export analysis as Markdown |

## Tech Stack

`Python` · `Streamlit` · `Google Gemini API` · `PyPDF2` · `Docker`
